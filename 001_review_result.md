## 상세 설명

### 문제점 1: 미사용 가져오기 (app.js, Line 5)

- `getTimeForTimezone`을 가져왔지만 app.js에서 사용하지 않음
- 시간은 로컬 시스템 시간을 사용하며, 타임존별 시간이 아님

### 문제점 2: 미완성 타임존 함수 (api.js, Line 141-161)

- `getTimezone` 함수가 좌표에서 올바르게 타임존을 계산하지 않음
- 현재 'UTC'만 반환
- 타임존 인식 시간 표시에 영향

### 문제점 3: 이벤트 핸들러 (app.js, Line 62-68)

- `globe.controls()`이 즉시 사용 가능하지 않을 수 있음
- 이벤트 연결이 조용히 실패할 수 있음
